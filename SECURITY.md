# Security Policy

## Supported Versions

Aegis Core is currently alpha software. Security fixes are accepted for the default branch and the latest tagged release.

## Reporting a Vulnerability

Please use GitHub private vulnerability reporting or open a private security advisory for this repository. Do not include live claim URLs, production databases, `.env` files, relay credentials, SMTP credentials, S3 credentials, packet keys, or personal estate data in public issues.

Include:

- Affected version or commit SHA
- Reproduction steps against a local test instance
- Expected and actual behavior
- Whether credentials, claim tokens, packets, or encrypted fields may be exposed

## Disclosure

Public disclosure should wait until a fix is available and users have had reasonable time to upgrade. This repository is AGPL-3.0-only; source recipients should receive security fixes under the same license terms.
